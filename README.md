# Java Web Project

An Die Musik(음악에게) 공연 예매 시스템 

## 제안 배경

- 코로나로 인한 공연예술계의 침체를 극복하고자 대전 지역의 시립 예술단 공연 정보를 보여주고 예매 기능을 구현하며 회원들의 커뮤니티 공간을 제공하기 위해 기획
-  Java기반의 웹 프로젝트
- CRUD에 중점을 두어 기능을 구현
- 대덕인재개발원 2차 중급 Project

## 개발 기간 & 구성원

- 개발 기간 : 21 / 05 / 20 ~ 21/ 06 / 05 
- 구성원 : 김현성(PL) , 김찬미(DA), 오태훈(TA), 이치우(UA)

## 주요 기능

- 회원가입 & 로그인
- 공연 정보
- 공연 예매 & 결제
- 마이페이지
- 단체 소개
- 자유게시판 CRUD
- Q&A CRUD
- 공지사항 CRUD
- 공연 후기

## My Role

- 공연 정보 CRUD
- 공연 예매 & 결제
-  마이페이지
- 단체소개
